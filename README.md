# Прибыль продаж

Простое Android-приложение для учета продаж по сменам.

## Что умеет

- автоматически создает смену на текущую дату телефона;
- принимает наименование товара, закуп и цену продажи;
- считает прибыль по формуле `цена продажи - закуп`;
- хранит список продаж за смену;
- учитывает ежедневную аренду;
- закрывает смену и считает:
  - общую прибыль продаж;
  - прибыль после вычета аренды;
  - зарплату продавца как `60%` от суммы после аренды;
- показывает журнал смен.

## Как открыть

Откройте папку проекта в Android Studio и запустите приложение на телефоне или эмуляторе.

APK автоматически собирается через GitHub Actions после изменений в `main`.

Если Android пишет, что приложение конфликтует с другим пакетом, удалите старую установку один раз и поставьте APK версии 1.1.1 или новее. Дальнейшие обновления будут ставиться поверх, потому что проект использует постоянный ключ подписи.
